# VK-бот «Жадный Мир»

Асинхронный бот сообщества VK для реестра карт, анкет персонажей, Шакеев и
вспомогательного кубика. Интерфейсы физически разделены: пошаговые сценарии ЛС
находятся в `bot/handlers/dm/`, команды беседы - в `bot/handlers/chat/`.

## Возможности MVP

- ЛС: выбор одной из своих анкет, постраничные реестры карт и анкет, поиск
  карты, перевод Шакеев от имени выбранного персонажа.
- Админ-панель в ЛС: создание анкет по шаблону, CRUD карт, статы и рейтинг,
  начисление/списание Шакеев.
- AI-импорт в админ-панели: исходную анкету можно прислать несколькими
  сообщениями и дополнить изображениями внешности или книги. Модель переносит
  только предоставленные сведения без дополнений, результат показывается для
  проверки и записывается в базу лишь после подтверждения. Отдельный сценарий
  оформляет идеи Контуров.
- Один VK ID может владеть несколькими анкетами; новые анкеты добавляет только
  администратор.
- Администратор открывает персонажа из реестра либо находит его по имени,
  `#ID` анкеты или VK ID владельца. Если одному VK ID принадлежат несколько
  персонажей, бот показывает отдельный выбор и не подставляет первого попавшегося.
- В карточке карты и анкете администратор получает быстрые кнопки редактирования
  и удаления. Удаление выполняется только после отдельного подтверждения;
  для обычных пользователей реестры и собственные анкеты доступны только для чтения.
- Кнопка `Создать бэкап БД` в админ-панели делает согласованный SQLite-снимок,
  проверяет его целостность и отправляет администратору `.db`-файл документом VK.
- Добавление карты начинается с выбора типа и заполнения многострочного
  шаблона: Особая, Заклинание, Обычная, Контурная или ГеймМастерская.
- Беседа: `?!карта`, `?!профиль`, `?!кубик`, `?!помощь`.
- Реальные копии карт хранятся в `CardOwnership`; выдача учитывает лимит
  преобразований, изъятие освобождает лимит.
- Каждое движение Шакеев записывается в журнал и может быть сверено с балансом.

## Установка

Нужен Python 3.11 или новее.

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
Copy-Item .env.example .env
```

Заполните в `.env` токен сообщества, ID группы и VK ID администраторов. Для
кнопок `AI → Анкета` и `AI → Контур` также добавьте ключ AI Tunnel:

```dotenv
AITUNNEL_API_KEY=sk-aitunnel-xxx
AITUNNEL_MODEL=gemini-3.1-flash-lite
```

Присланный в этих сценариях текст передаётся внешнему сервису AI Tunnel и
выбранной модели Google Gemini. Затем
создайте схему и запустите бота:

```powershell
python -m alembic upgrade head
python run.py
```

Для PostgreSQL достаточно заменить `DATABASE_URL`, например на
`postgresql://user:password@localhost/zhadny_mir`, и снова выполнить миграции.

## Команды беседы

```text
?!карта Ясень
?!профиль
?!профиль Ава
?!кубик
?!кубик 6
?!кубик 1 20
?!помощь
```

Кубик - последний довод для тупиковой сцены, а не замена тексту, позиции и
подготовке персонажей.

## Админские команды в ЛС

Основные операции запускаются постоянной клавиатурой VK под полем ввода.
Дополнительно доступны:

```text
?!выдать <название карты> <имя персонажа>
?!забрать <название карты> <имя персонажа>
```

Права для всего админского набора проверяет общее правило на лейблере по
`ADMIN_VK_IDS`, а не каждый обработчик отдельно.
